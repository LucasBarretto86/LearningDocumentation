Bootstrap - Aula 03 - Badges E Alerts

Badges e Alerts são elementos visuais que servem para apresentar informações através de mensagens ou stickers na página. Estes tem uma relação importante com o sistemas de cores do Bootstrap.

Aula de colorização no Bootstrap

Badges

.badge e badge-"cor"

Os elementos de classe badge são elementos inline, por tanto, normalmente os elementos span são utilizados como badges, é compreensível pois o elemento span é muito versátil.

Alerts

.alert e .alert-"cor"

Já os elementos da classe .alert são elementos em bloco, usa-se normalmente o elemento div para a criação de alerts.
