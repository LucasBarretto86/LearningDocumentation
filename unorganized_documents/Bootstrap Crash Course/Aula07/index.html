Bootstrap - Aula 07 - Modal, caixas de diálogos

Modais no Bootstrap permitem a customização de caixa de dialogo, é um componente que usa tanto HTML, CSS como Javascript, são bem interessantes, mas, possuem detalhes que devem ser analisados com maior cuidado, não se trata apenas de vincular classes pre-definidas, requer também o desenvolvimento dos elementos deste componente, por tanto, iremos abordar nessa aula os detalhes para a confecção de modais por passos.

Trigger, elemento de disparo do modal

O primeiro passo será definir um elemento que servirá para disparar o nosso modal.

Este gatilho deverá possuir duas propriedades que apontem para a criação/exibição de um modal, data-toggle que declara o tipo de elemento que será exibido, data-target que indica o id que iremos estabelecer para o main container do nosso modal:

Main container do modal

Depois devemos criar um elemento div que representará o nosso modal como um todo, esse elemento recebe a classe .modal, também devemos definir um id para este elemento, é dentro deste elemento que criarmos os componentes do nosso modal:

Sub container que define o tipo de modal

Dentro do container principal devemos criar outra div que representará um sub-container com uma classe .modal-dialog que determina o tipo de modal que iremos compor.

Conteúdo do modal

Dentro do container/tipo devemos criar uma nova div que irá agregar 3 blocos para a composição do nosso modal, bloco header, bloco body, bloco footer.

Bloco header

Este bloco é aonde o titulo do modal será exibido, assim como definirá o botões de comportamento do modal, neste exemplo um simples botão de fechar.

O botão de fechar precisa definir a classe .close e também atribuir uma propriedade data-dismiss apontando para o elemento que será fechado.

Bloco body

Este bloco tem o propósito de apresentar o conteúdo principal do modal

Bloco footer

Este bloco serve principalmente para apresentar os botões de ação referentes ao modal.

Neste exemplo teremos també um simples botão para fechar o modal.
