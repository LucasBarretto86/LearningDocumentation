Bootstrap - Aula 06 - Barras de Progresso

O Bootstrap também dispoe de barras de progresso customizadas e que aceitam colorização, estilização, animação, como veremos a seguir.

Barra de progresso padrão

Para criar a barra de progresso primeiramente criamos uma div com a classe .progress, essa primeira div representa o recipiente do progresso e dentro desta div nós criamos outra div que recebe a classe .progress-bar que será a barra de progresso em si, esta div especialista manifestará o processo, se usada juntamente com um javascript.
