Bootstrap - Aula 02 - Tables

Classes para tables

O Bootstrap também disponibiliza classes para se trabalhar de fora simplificada com tabelas.

Neste exemplo usaremos uma divs com a classe .container para que o efeito sobre tabelas fique mais claro, pois, uma vez que a proposta dos grids a modularizadão, os elementos presentes em cada div tendem a preencher completamente a div.

Cada container possuirá uma tabla em que será aplicada uma variação/combinação de classes do Bootstrap.
