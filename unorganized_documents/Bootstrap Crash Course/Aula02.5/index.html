Bootstrap - Aula 02.5 - Colorização

Colorização

Colorização no Bootstrap segue um padrão interessante, o nome das "cores" que na verdade são templates são específicos; default, success, info, Warning, danger, primary, secondary, active, light, dark.

No entanto, nem todos os elementos irão ter uma representação para todas as cores, quando isso ocorre o elemento simplesmente mantém sua cor padrão.

Muitos dos elementos possuem classes auxiliares de colorização e estilização, como por exemplo um elemento button, pois uma classe auxiliar de colorização .btn-"cor", ou como um elemento table que possui a classe .table-"cor". No entanto, existe uma classe que representa de forma geral um background colorido para os elementos que não possui uma classe de colorização especifica essa classe é .bg-"cor", abaixo segue a aplicação dessas cores em alguns elementos:
