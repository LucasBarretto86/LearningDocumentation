Rows e Columns, linhas e colunas

O sistema de Grid do Bootstrap é muito coeso para a criação de layouts, constituído por uma 'flexbox' predefine o tamanho dos containers de acordo com colunas de tamanho pre estipulados, podendo este ter até 12 colunas.

Para que as linhas e colunas sejam implementadas nos containers existem divs e classes adicionais que devemos acrescentar

.row
Define o elemento que representará a linha, normalmente é usada uma div.
.col-*-**
  1. * representa o tamanho da tela do device para o qual a coluna será direcionado.
    .col-**
    Colunas colapsarão automaticamente, pois o container width é automático;para telas "(&tl 576px) com portrait view.
    .col-sm-**
    Colunas colapsarão acima do tamanho do container, container width 540px; para telas "(>= 576px) horizontal.
    .col-md-**
    Colunas colapsarão acima do tamanho do container, container width 720px; para telas "(>= 768px) horizontal.
    .col-lg-**
    Colunas colapsarão acima do tamanho do container, container width 960px; para telas "(>= 992px) horizontal (laptop).
    .col-xl-**
    Colunas colapsarão acima do tamanho do container, container width 1140px; para telas "(>= 1200px) horizontal (laptop e desktop).
  2. ** representa o número de colunas que deverão ser apresentadas com o limite de até 12 colunas.

Exemplo de um Grid com uso de linhas e colunas
